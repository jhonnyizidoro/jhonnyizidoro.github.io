Meus Projetos

Por ainda ser novo na área de desenvolvimento, não tenho muitos projetos realizados, mas mesmo assim estou sempre envolvido no desenvolvimento de alguma ferramenta, seja profissionalmente ou para aprender.

GrandPrix Automóveis

Site da loja GrandPrix onde ela anuncia seus veículos. Esse site representou um grande desafio, pois uma mas maiores preocupações que tivemos foi otimizar a performance dele para que a ele tivesse uma nota boa no Google PageSpeed Insight. Isso mudou toda a dinâmica do desenvolvimento. O site é totalmente responsivo e foi feito utilizando tecnologias modernas.

PROTECA

Site para divulgação de notícias feito em um projeto da Universidade Federal do Paraná. O conteúdo do site tem o intuito de prevenir o aliciamento de crianças e adolescentes pela internet. O grande desafio desse projeto foi a responsabilidade que senti, pois além de ser o meu primeiro projeto importante, o projeto representa uma causa muito nobre, que é a de proteger nossas crianças e adolescentes.

Sistema de inscrição

Sistema de inscrição para um evento que aconteceu ainda na primeira semana em que participei do projeto PROTECA. Me diverti muito fazendo esse projeto. Eu precisava salvar os registros de quem se cadastrava, mas não tinha onde hospedar o sistema. Então decidi fazer o projeto com Vue JS e Firebase, assim consegui hospedar o sistema no GitHub como uma página estática.
